### Тестовое задание Java ЕЕ

#### Бибаев Руслан

### Описание

REST API для отслеживания почтовых отправлений

### Требования

* Java 17
* Gradle 8.0

### Запуск

Для запуска выполните команду
```
gradlew bootRun
```

Или
```
./gradlew bootRun
```

### Сборка

Для сборки проекта выполните команду
```
gradlew bootWar
```

Или
```
./gradlew bootWar
```
